# 🏆 UEFA Champions League - Analytics & Standings Module
![Vista previa del sistema](estr-und3/Captura%20de%20pantalla%202026-04-22%20161936.png)

¡Bienvenido al Módulo de Gestión de Datos Oficial de la UEFA Champions League! Este software permite el seguimiento en tiempo real de las estadísticas de competición, diseñado con una interfaz profesional y multimedia.

## ✨ Características Principales
* **Gestión de Datos:** Registro dinámico de Goles, Partidos y Puntos.
* **Algoritmo de Clasificación:** Tabla automatizada con lógica de desempate por diferencia de goles.
* **Interfaz Pro:** Visualización con estados de clasificación (Verde: Clasificado / Rojo: Eliminado).
* **Experiencia Multimedia:** Integración de audio nativo (Himno oficial) y branding visual.

## 🚀 Tecnologías Utilizadas
* **Python 3.x**
* **Tkinter** (Interfaz Gráfica)
* **Winsound** (Gestión de Audio en Windows)
* **Threading** (Procesamiento en segundo plano para audio)

## 🛠️ Instalación y Uso
1. Clona este repositorio.
2. Asegúrate de tener los archivos `logo.jpg` y `himno.wav` en la carpeta raíz.
